# macOS DMG 打包说明

## 前置要求

| 要求 | 说明 |
|------|------|
| 操作系统 | macOS 12 (Monterey) 或更高版本 |
| 芯片 | Apple Silicon（M1 / M2 / M3），即 arm64 架构 |
| Python | 3.10+，**arm64 原生版**（从 [python.org](https://www.python.org/downloads/macos/) 下载，不要用 Rosetta） |
| 磁盘空间 | 构建过程约需 1 GB 临时空间 |
| 网络 | 需要联网下载依赖包 |

> **验证 Python 架构**：打开终端，运行 `python3 -c "import platform; print(platform.machine())"`，应输出 `arm64`。

---

## 一键构建

把项目克隆到 Mac 后，在项目根目录执行：

```bash
chmod +x build_mac.sh
./build_mac.sh
```

脚本会自动完成以下步骤：
1. 创建隔离构建虚拟环境（`.build_venv/`）
2. 安装核心依赖（Flask、SQLAlchemy、python-docx、openai、python-dotenv、PyInstaller）
3. 调用 PyInstaller 按 `exam_system.spec` 打包
4. 用 `hdiutil` 生成 DMG 文件

**产物路径**：

```
dist/试题管理系统_v1.7.0_arm64.dmg
```

---

## 安装与使用

1. 双击打开 DMG 文件
2. 将「**试题管理系统**」拖入「**Applications**」文件夹
3. 双击启动

> **首次启动提示**：macOS 可能弹出「无法验证开发者」提示。
> 前往「系统设置 → 隐私与安全性」，点击「仍要打开」即可。

启动后约 2 秒，系统浏览器会自动打开 `http://127.0.0.1:505x`，即可使用。

---

## 数据目录

所有用户数据（题库、数据库、导出文件、AI 知识库）存储在：

```
~/Library/Application Support/试题管理系统/
├── exam_system.db       # 题库数据库
├── ds_knowledge.db      # DeepSeek 知识图谱数据库
├── .env                 # API Key 配置
├── uploads/             # 图片附件
├── exports/             # 导出的 Word 文件
└── rag_uploads/         # 上传的文档（DS 模式）
```

**卸载**：删除 `/Applications/试题管理系统.app` 即可。数据目录不会自动删除，如需清除请手动删除上述路径。

## 常见问题

**Q：构建时提示「Python 不是 arm64」怎么办？**
从 [python.org](https://www.python.org/downloads/macos/) 下载 macOS 版（Universal2 或 arm64），安装后重新打开终端再执行脚本。

**Q：应用启动后浏览器没有自动打开？**
手动在浏览器输入 `http://127.0.0.1:5050`（或 5051、5052，端口从 5050 开始依次探测）。

**Q：DeepSeek API Key 在哪里填写？**
启动后进入「知识图谱」标签页，点击「API 配置」，填入 Key 后保存。Key 会写入 `~/Library/Application Support/试题管理系统/.env`，重启后仍有效。

**Q：如何同时支持 Intel Mac？**
编辑 `exam_system.spec`，将 `target_arch='arm64'` 改为 `target_arch='universal2'`，然后在 Intel Mac 或 Rosetta 环境下重新构建。注意：universal2 要求所有二进制依赖均提供 universal2 wheel，可能需要额外调试。

## 手动构建（高级）

如需自定义，可跳过脚本手动操作：

```bash
# 1. 创建并激活虚拟环境
python3 -m venv .build_venv
source .build_venv/bin/activate

# 2. 安装依赖
pip install flask flask-sqlalchemy "python-docx>=1.1" lxml openai python-dotenv pyinstaller

# 3. 打包
pyinstaller exam_system.spec

# 4. 生成 DMG
mkdir -p dist/.dmg_staging
cp -r "dist/试题管理系统.app" dist/.dmg_staging/
ln -s /Applications dist/.dmg_staging/Applications
hdiutil create -volname "试题管理系统" -srcfolder dist/.dmg_staging \
    -ov -format UDZO "dist/试题管理系统_v1.7.0_arm64.dmg"
rm -rf dist/.dmg_staging

deactivate
```
